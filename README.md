# Text Classification (python_nb_sklearn_gaussian_naive_bayes)
Repository containing attempts of text classification using Naive Bayes method.

# Installation
**Requirements**
- `sklearn`
- `pandas`
- `numpy`
- `jupyter`

```sh
pip install jupyter pandas numpy sklearn
```
